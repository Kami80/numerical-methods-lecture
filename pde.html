01

PDE Classification Foundation

Before choosing a numerical method, identify the physical type of the PDE, the required boundary data, and whether the solution evolves in time or represents a steady field.

General second-order linear PDE in two variables
\[ A\frac{\partial^2 u}{\partial x^2} +B\frac{\partial^2 u}{\partial x\partial y} +C\frac{\partial^2 u}{\partial y^2} +D\frac{\partial u}{\partial x} +E\frac{\partial u}{\partial y} +Fu=G \]

The classification depends mainly on the discriminant:

\[ \Delta = B^2-4AC \]
TypeConditionTypical equationPhysical meaningCommon numerical methods
Elliptic\(B^2-4AC<0\)\(u_{xx}+u_{yy}=0\)Steady heat conduction, potential flow, electrostaticsFDM, Gauss-Seidel, SOR/PSOR, FEM
Parabolic\(B^2-4AC=0\)\(u_t=\alpha u_{xx}\)Diffusion and unsteady heat transferExplicit, implicit, Crank-Nicolson, ADI, MOL
Hyperbolic\(B^2-4AC>0\)\(u_{tt}=c^2u_{xx}\)Waves, vibrations, acousticsCentral difference, Lax-Wendroff, characteristics
Initial condition: needed when the PDE evolves in time. Example: \(T(x,0)=T_0(x)\).
Boundary condition: needed at physical boundaries. Example: fixed wall temperature, heat flux, or convection.

Boundary condition types

TypeMathematical formEngineering interpretation
Dirichlet\(u=a\)Specified temperature, concentration, or displacement
Neumann\(\partial u/\partial n=q\)Specified heat flux or mass flux
Robin\(a u+b\partial u/\partial n=c\)Convection boundary, radiation approximation, mixed transfer
02

Finite Difference Grid Setup Stencil Logic

The finite difference method replaces derivatives by algebraic approximations at grid nodes. This transforms a PDE into equations that can be solved by matrix methods or iteration.

Forward difference
\[u_x\big|_i\approx \frac{u_{i+1}-u_i}{\Delta x}\]
Backward difference
\[u_x\big|_i\approx \frac{u_i-u_{i-1}}{\Delta x}\]
Central first derivative
\[u_x\big|_i\approx \frac{u_{i+1}-u_{i-1}}{2\Delta x}\]
Central second derivative
\[u_{xx}\big|_i\approx \frac{u_{i-1}-2u_i+u_{i+1}}{\Delta x^2}\]
Two-dimensional Laplacian, uniform grid
\[ \nabla^2 u\big|_{i,j} \approx \frac{u_{i-1,j}-2u_{i,j}+u_{i+1,j}}{\Delta x^2} + \frac{u_{i,j-1}-2u_{i,j}+u_{i,j+1}}{\Delta y^2} \]

For \(\Delta x=\Delta y=h\):

\[ \nabla^2u\approx\frac{u_E+u_W+u_N+u_S-4u_P}{h^2} \]
03

Laplace Equation Steady Field

Laplace equations describe steady systems with no internal source term. The value at an interior node is the average of its four neighboring nodes.

Five-point Laplace update
\[ \frac{T_E+T_W+T_N+T_S-4T_P}{h^2}=0 \qquad\Longrightarrow\qquad T_P=\frac{T_E+T_W+T_N+T_S}{4} \]
🧊
Worked Example — 2×2 Interior Plate

Solve \(\nabla^2T=0\) on a square plate with four interior nodes. The top boundary is \(100^\circ C\); the left, right, and bottom boundaries are \(0^\circ C\).

1Write the four node equations

Let the bottom row be \(T_{11},T_{21}\) and the top interior row be \(T_{12},T_{22}\).

\[ \begin{aligned} 4T_{11}-T_{21}-T_{12}&=0\\ -T_{11}+4T_{21}-T_{22}&=0\\ -T_{11}+4T_{12}-T_{22}&=100\\ -T_{21}-T_{12}+4T_{22}&=100 \end{aligned} \]
2Solve the linear system

By symmetry, \(T_{11}=T_{21}=a\) and \(T_{12}=T_{22}=b\).

Bottom equation: \(4a-a-b=0\Rightarrow 3a=b\)
Top equation: \(-a+4b-b=100\Rightarrow -a+3b=100\)
Substitute \(b=3a\): \(-a+9a=100\Rightarrow a=12.5\)
Therefore \(b=37.5\)
NodeValueInterpretation
\(T_{11}\)12.5000Bottom-left interior node
\(T_{21}\)12.5000Bottom-right interior node
\(T_{12}\)37.5000Top-left interior node
\(T_{22}\)37.5000Top-right interior node
▶ MATLAB Code
Laplace equation by SOR
%% Laplace equation on a rectangular grid using Gauss-Seidel SOR
clc; clear; close all;

Nx = 21; Ny = 21;
T = zeros(Ny,Nx);

T(end,:) = 100;       % top boundary
T(1,:)   = 0;         % bottom boundary
T(:,1)   = 0;         % left boundary
T(:,end) = 0;         % right boundary

omega = 1.6;
tol = 1e-8;
maxIter = 20000;

for iter = 1:maxIter
    T_old = T;

    for j = 2:Ny-1
        for i = 2:Nx-1
            T_gs = 0.25*(T(j,i-1) + T(j,i+1) + T(j-1,i) + T(j+1,i));
            T(j,i) = (1-omega)*T(j,i) + omega*T_gs;
        end
    end

    err = max(abs(T(:)-T_old(:)));
    if err < tol
        break
    end
end

fprintf('Iterations = %d\n', iter)
contourf(T,20); colorbar; axis equal tight;
title('Laplace Equation Solution by SOR');
04

Poisson Equation Source Term

Poisson equations include internal generation or consumption. In heat transfer, this represents volumetric heat generation inside a solid.

Poisson update for \(\nabla^2T=-S\)
\[ \frac{T_E+T_W+T_N+T_S-4T_P}{h^2}=-S \qquad\Longrightarrow\qquad T_P=\frac{T_E+T_W+T_N+T_S+h^2S}{4} \]
🔥
Worked Example — Uniform Source on a 2×2 Grid

Solve \(\nabla^2T=-36\) on a unit square with all boundaries fixed at zero. Use 2 interior nodes in each direction, so \(h=1/3\).

1Compute the source contribution
\[ h^2S=\left(\frac{1}{3}\right)^2(36)=4 \]

Each interior equation has the form \(4T_P-T_E-T_W-T_N-T_S=4\).

2Use symmetry

All four interior nodes are identical by symmetry, so \(T_P=a\).

\(4a-a-a=4\)
\(2a=4\Rightarrow a=2\)
The source term raises the interior temperature above the zero-temperature boundary. If the source were absent, the full solution would be zero everywhere.
▶ MATLAB Code
Poisson equation matrix assembly
%% Poisson equation on a square plate: del2(T) = -S
clc; clear; close all;

N = 20;
L = 1;
h = L/(N+1);
S = 36;

A = sparse(N*N,N*N);
b = zeros(N*N,1);

idx = @(i,j) (j-1)*N + i;

for j = 1:N
    for i = 1:N
        p = idx(i,j);
        A(p,p) = 4;
        b(p) = h^2*S;

        if i > 1, A(p,idx(i-1,j)) = -1; end
        if i < N, A(p,idx(i+1,j)) = -1; end
        if j > 1, A(p,idx(i,j-1)) = -1; end
        if j < N, A(p,idx(i,j+1)) = -1; end
    end
end

Tvec = A\b;
T = reshape(Tvec,N,N);

surf(T); shading interp; colorbar;
title('Poisson Equation: Interior Source Term');
05

Explicit Heat Equation FTCS

The explicit method is easy to program because the new time level is calculated directly from the old time level. Its disadvantage is conditional stability.

Forward-Time Central-Space method
\[ \frac{T_i^{n+1}-T_i^n}{\Delta t} =\alpha\frac{T_{i-1}^n-2T_i^n+T_{i+1}^n}{\Delta x^2} \] \[ T_i^{n+1}=rT_{i-1}^n+(1-2r)T_i^n+rT_{i+1}^n, \qquad r=\frac{\alpha\Delta t}{\Delta x^2} \]
Stability requirement: for the one-dimensional FTCS heat equation, use \(r\le 0.5\). If \(r>0.5\), numerical oscillations grow even when the physical solution should be smooth.
⏩
Worked Example — Two Explicit Time Steps

Rod length \(L=1\), \(\Delta x=0.25\), \(\alpha=1\), \(\Delta t=0.02\). Boundary temperatures are \(T_0=100\) and \(T_4=50\). Initial interior values are \(T_1^0=T_2^0=T_3^0=20\).

1Check stability
\[ r=\frac{1(0.02)}{(0.25)^2}=\frac{0.02}{0.0625}=0.32<0.5 \]
2Calculate the first time step
\(T_1^1=0.32(100)+0.36(20)+0.32(20)=45.6000\)
\(T_2^1=0.32(20)+0.36(20)+0.32(20)=20.0000\)
\(T_3^1=0.32(20)+0.36(20)+0.32(50)=29.6000\)
3Calculate the second time step
\(T_1^2=0.32(100)+0.36(45.6)+0.32(20)=54.8160\)
\(T_2^2=0.32(45.6)+0.36(20)+0.32(29.6)=31.2640\)
\(T_3^2=0.32(20)+0.36(29.6)+0.32(50)=33.0560\)
▶ MATLAB Code
Explicit FTCS heat solver
%% Explicit FTCS method for 1D heat equation
clc; clear; close all;

alpha = 1;
L = 1;
dx = 0.25;
dt = 0.02;
r = alpha*dt/dx^2;

x = 0:dx:L;
Nt = 20;
T = 20*ones(length(x),1);
T(1) = 100;
T(end) = 50;

if r > 0.5
    error('Unstable FTCS grid: r must be <= 0.5')
end

history = zeros(length(x),Nt+1);
history(:,1) = T;

for n = 1:Nt
    Told = T;
    for i = 2:length(x)-1
        T(i) = r*Told(i-1) + (1-2*r)*Told(i) + r*Told(i+1);
    end
    history(:,n+1) = T;
end

plot(x,T,'-o','LineWidth',2); grid on;
xlabel('x'); ylabel('Temperature');
title('Explicit FTCS Heat Equation Result');

Implicit Heat Equation Backward Euler

The implicit method evaluates the spatial derivative at the new time level. It requires solving a linear system at every time step but is unconditionally stable for the linear heat equation.

Backward Euler in time, central difference in space
\[ -rT_{i-1}^{n+1}+(1+2r)T_i^{n+1}-rT_{i+1}^{n+1}=T_i^n \]
⏪
Worked Example — One Implicit Step

Use the same rod and initial condition as before, but take \(\Delta t=0.0625\). Then \(r=1\).

Linear system
\[ \begin{bmatrix} 3&-1&0\\ -1&3&-1\\ 0&-1&3 \end{bmatrix} \begin{bmatrix}T_1^1\\T_2^1\\T_3^1\end{bmatrix} = \begin{bmatrix}120\\20\\70\end{bmatrix} \]
UnknownValue after one implicit step
\(T_1^1\)51.9048
\(T_2^1\)35.7143
\(T_3^1\)35.2381

Backward Euler heat solver
%% Backward Euler implicit method for 1D heat equation
clc; clear; close all;

alpha = 1;
L = 1;
dx = 0.25;
dt = 0.0625;

b = Told;
b(1) = b(1) + r*Tleft;
b(end) = b(end) + r*Tright;

Tnew = A\b;
disp(Tnew)

Crank–Nicolson Method Second Order Time

Crank–Nicolson averages the explicit and implicit spatial approximations. It is more accurate in time than backward Euler and is widely used in diffusion problems.

Crank–Nicolson stencil
\[ -\frac r2T_{i-1}^{n+1}+(1+r)T_i^{n+1}-\frac r2T_{i+1}^{n+1} = \frac r2T_{i-1}^n+(1-r)T_i^n+\frac r2T_{i+1}^n \]
⚖️
Worked Example — One Crank–Nicolson Step

Use the same 3 interior nodes as the implicit example with \(r=1\), \(T_0=100\), \(T_4=50\), and initial interior values equal to 20.

Crank–Nicolson system
\[ \begin{bmatrix} 2&-0.5&0\\ -0.5&2&-0.5\\ 0&-0.5&2 \end{bmatrix} \begin{bmatrix}T_1^1\\T_2^1\\T_3^1\end{bmatrix} = \begin{bmatrix}110\\20\\60\end{bmatrix} \]
UnknownValue after one CN step
\(T_1^1\)63.9286
\(T_2^1\)35.7143
\(T_3^1\)38.9286
▶ MATLAB Code
Crank–Nicolson heat solver

alpha = 1;
dx = 0.25;
dt = 0.0625;

b = B*Told;
b(1) = b(1) + r*Tleft;
b(end) = b(end) + r*Tright;

Tnew = A\b;
disp(Tnew)

ADI Method Alternating Direction Implicit

ADI is designed for two-dimensional parabolic PDEs. It splits one full time step into two half-steps: first solve tridiagonal systems in the x-direction, then solve tridiagonal systems in the y-direction.

2D heat equation
\[ \frac{\partial T}{\partial t} =\alpha\left(\frac{\partial^2T}{\partial x^2}+\frac{\partial^2T}{\partial y^2}\right) \] \[ r_x=\frac{\alpha\Delta t}{\Delta x^2},\qquad r_y=\frac{\alpha\Delta t}{\Delta y^2} \]

Peaceman–Rachford ADI splitting

1First half-step: x-direction implicit, y-direction explicit
\[ -\frac{r_x}{2}T_{i-1,j}^* +(1+r_x)T_{i,j}^* -\frac{r_x}{2}T_{i+1,j}^* = \frac{r_y}{2}T_{i,j-1}^n +(1-r_y)T_{i,j}^n +\frac{r_y}{2}T_{i,j+1}^n \]

For each fixed row \(j\), solve a tridiagonal system along the x-direction.

2Second half-step: y-direction implicit, x-direction explicit
\[ -\frac{r_y}{2}T_{i,j-1}^{n+1} +(1+r_y)T_{i,j}^{n+1} -\frac{r_y}{2}T_{i,j+1}^{n+1} = \frac{r_x}{2}T_{i-1,j}^* +(1-r_x)T_{i,j}^* +\frac{r_x}{2}T_{i+1,j}^* \]

For each fixed column \(i\), solve a tridiagonal system along the y-direction.

Why ADI is useful: a fully implicit 2D method solves one large matrix system. ADI replaces that large system with many small tridiagonal systems, making each time step much cheaper.
↔️
Worked Example — One ADI Step on a 2×2 Interior Grid

Consider a square plate with 2×2 interior nodes. All boundaries are zero. Initial interior temperature is \(100^\circ C\) at all four nodes. Let \(r_x=r_y=0.5\).

1Build the x-sweep system

For each row, the tridiagonal matrix is

\[ A_x=\begin{bmatrix}1.5&-0.25\\-0.25&1.5\end{bmatrix} \]

For either row, the right-hand side is

\[ b_x=0.25(0)+0.5(100)+0.25(100)=75 \] \[ \begin{bmatrix}1.5&-0.25\\-0.25&1.5\end{bmatrix} \begin{bmatrix}T_1^*\\T_2^*\end{bmatrix} = \begin{bmatrix}75\\75\end{bmatrix} \Rightarrow T_1^*=T_2^*=60 \]
2Build the y-sweep system

For each column, the y-sweep matrix is the same:

\[ A_y=\begin{bmatrix}1.5&-0.25\\-0.25&1.5\end{bmatrix} \]

The right-hand side is

\[ b_y=0.25(0)+0.5(60)+0.25(60)=45 \] \[ A_y\begin{bmatrix}T_1^{n+1}\\T_2^{n+1}\end{bmatrix} =\begin{bmatrix}45\\45\end{bmatrix} \Rightarrow T_1^{n+1}=T_2^{n+1}=36 \]

Initial100.0000Hot interior, cold boundary
After x-sweep60.0000Intermediate half-step \(T^*\)

▶ MATLAB Code
ADI 2D heat solver
%% ADI method for 2D heat equation
clc; clear; close all;

Nx = 20; Ny = 20;
alpha = 1;
dx = 1/(Nx+1);
dy = 1/(Ny+1);
dt = 0.0005;
rx = alpha*dt/dx^2;
ry = alpha*dt/dy^2;

T = 100*ones(Ny,Nx);       % interior initial temperature
nt = 50;

Ax = diag((1+rx)*ones(Nx,1)) + diag(-(rx/2)*ones(Nx-1,1),1) + diag(-(rx/2)*ones(Nx-1,1),-1);
Ay = diag((1+ry)*ones(Ny,1)) + diag(-(ry/2)*ones(Ny-1,1),1) + diag(-(ry/2)*ones(Ny-1,1),-1);

for n = 1:nt
    Tstar = zeros(Ny,Nx);

    % x-implicit sweep: solve one tridiagonal system per row
    for j = 1:Ny
        rhs = (1-ry)*T(j,:)';
        if j > 1, rhs = rhs + (ry/2)*T(j-1,:)'; end
        if j < Ny, rhs = rhs + (ry/2)*T(j+1,:)'; end
        Tstar(j,:) = (Ax\rhs)';
    end

    % y-implicit sweep: solve one tridiagonal system per column
    for i = 1:Nx
        rhs = (1-rx)*Tstar(:,i);
        if i > 1, rhs = rhs + (rx/2)*Tstar(:,i-1); end
        if i < Nx, rhs = rhs + (rx/2)*Tstar(:,i+1); end
        T(:,i) = Ay\rhs;
    end
end

imagesc(T); axis equal tight; colorbar;
title('2D Heat Equation Solved by ADI');

Method of Lines PDE → ODE System

The Method of Lines discretizes only the spatial variable first. The result is a system of ordinary differential equations in time, which can be solved using standard ODE solvers.

Heat equation transformed into an ODE system
\[ \frac{dT_i}{dt}=\alpha\frac{T_{i-1}-2T_i+T_{i+1}}{\Delta x^2}, \qquad i=1,2,\ldots,N \] \[ \frac{d\mathbf{T}}{dt}=\mathbf{A}\mathbf{T}+\mathbf{b} \]
📈
Worked Example — Three Interior Nodes

For \(N=3\), \(\Delta x=0.25\), \(\alpha=1\), \(T_0=100\), and \(T_4=50\), derive the semi-discrete ODE system.

\[ \frac{d}{dt} \begin{bmatrix}T_1\\T_2\\T_3\end{bmatrix} = \frac1{0.25^2} \left[ \begin{bmatrix} -2&1&0\\ 1&-2&1\\ 0&1&-2 \end{bmatrix} \begin{bmatrix}T_1\\T_2\\T_3\end{bmatrix} + \begin{bmatrix}100\\0\\50\end{bmatrix} \right] \]

Since \(1/0.25^2=16\), the matrix system is stiff for small \(\Delta x\), so solvers such as ode15s are often preferred.

▶ MATLAB Code
Method of Lines with ode15s
%% Method of Lines for the 1D heat equation
clc; clear; close all;

alpha = 1;
L = 1;
N = 20;
dx = L/(N+1);
x = (1:N)'*dx;

Tleft = 100;
Tright = 50;
T0 = 20*ones(N,1);

rhs = @(t,T) heat_mol_rhs(T,alpha,dx,Tleft,Tright);
[t,T] = ode15s(rhs,[0 0.1],T0);

plot([0; x; L],[Tleft; T(end,:)'; Tright],'-o','LineWidth',2);
grid on; xlabel('x'); ylabel('Temperature');
title('Method of Lines Heat Equation Solution');

function dTdt = heat_mol_rhs(T,alpha,dx,Tleft,Tright)
    N = length(T);
    dTdt = zeros(N,1);
    for i = 1:N
        west = Tleft;
        east = Tright;
        if i > 1, west = T(i-1); end
        if i < N, east = T(i+1); end
        dTdt(i) = alpha*(west - 2*T(i) + east)/dx^2;
    end
end

Wave Equation Hyperbolic PDE

Wave equations require information from two time levels because the PDE contains a second derivative with respect to time.

Explicit central-difference wave scheme
\[ \frac{u_i^{n+1}-2u_i^n+u_i^{n-1}}{\Delta t^2} =c^2\frac{u_{i-1}^n-2u_i^n+u_{i+1}^n}{\Delta x^2} \] \[ u_i^{n+1}=2(1-\lambda^2)u_i^n-u_i^{n-1}+\lambda^2(u_{i-1}^n+u_{i+1}^n), \qquad \lambda=\frac{c\Delta t}{\Delta x} \]
CFL condition: for the standard one-dimensional explicit wave scheme, stability requires \(\lambda\le 1\).
🌊
Worked Example — First Two Wave Steps

Let \(c=1\), \(\Delta x=0.25\), \(\Delta t=0.2\), so \(\lambda=0.8\). Use fixed-end boundaries and \(u(x,0)=\sin(\pi x)\), with zero initial velocity.

Explicit wave equation solver
%% Explicit central-difference method for 1D wave equation
clc; clear; close all;

c = 1;
L = 1;
dx = 0.01;
dt = 0.008;
lambda = c*dt/dx;

if lambda > 1
    error('Unstable wave grid: c*dt/dx must be <= 1')
end

x = 0:dx:L;
Nt = 200;
u_old = sin(pi*x)';      % u(x,0)
v0 = zeros(size(x'));

u = u_old;
u(2:end-1) = u_old(2:end-1) + dt*v0(2:end-1) + 0.5*lambda^2*(u_old(1:end-2)-2*u_old(2:end-1)+u_old(3:end));

for n = 2:Nt
    u_new = zeros(size(u));
    for i = 2:length(x)-1
        u_new(i) = 2*(1-lambda^2)*u(i) - u_old(i) + lambda^2*(u(i-1)+u(i+1));
    end
    u_old = u;
    u = u_new;
end

plot(x,u,'LineWidth',2); grid on;
xlabel('x'); ylabel('u');
title('Explicit Wave Equation Solution');

PSOR Method Projected SOR

SOR accelerates Gauss-Seidel iteration by over-relaxation. PSOR adds a projection step, which is useful when the solution must satisfy an inequality constraint.

SOR and PSOR update
\[ T_P^{GS}=\frac{T_E+T_W+T_N+T_S+h^2S}4 \] \[ T_P^{SOR}=(1-\omega)T_P^{old}+\omega T_P^{GS}, \qquad 0<\omega<2 \] \[ T_P^{PSOR}=\max\left(\psi_P,\;T_P^{SOR}\right) \]

Here \(\psi_P\) is the lower bound or obstacle. If there is no obstacle, PSOR becomes ordinary SOR.

🔁
Worked Example — First SOR Sweep

Use the Laplace 2×2 interior plate from Section 3. Start with all interior nodes equal to zero and choose \(\omega=1.5\).

1Sweep in the order \(T_{11},T_{21},T_{12},T_{22}\)
\(T_{11}^{GS}=(0+0+0+0)/4=0\Rightarrow T_{11}^{new}=0\)
\(T_{21}^{GS}=(0+0+0+0)/4=0\Rightarrow T_{21}^{new}=0\)
\(T_{12}^{GS}=(0+0+100+0)/4=25\Rightarrow T_{12}^{new}=1.5(25)=37.5\)
\(T_{22}^{GS}=(37.5+0+100+0)/4=34.375\Rightarrow T_{22}^{new}=51.5625\)
SOR usually converges faster than Gauss-Seidel when \(\omega\) is chosen well. Too large a value may cause oscillation or divergence.
▶ MATLAB Code
PSOR / SOR iteration
%% PSOR / SOR iteration for Poisson-type PDEs
clc; clear; close all;

N = 30;
T = zeros(N+2,N+2);
T(end,:) = 100;        % top boundary
omega = 1.7;
tol = 1e-8;
maxIter = 10000;
obstacle = -inf(size(T));  % set finite values here for true projected SOR

for iter = 1:maxIter
    Told = T;

    for j = 2:N+1
        for i = 2:N+1
            gs = 0.25*(T(j,i-1)+T(j,i+1)+T(j-1,i)+T(j+1,i));
            sor = (1-omega)*T(j,i) + omega*gs;
            T(j,i) = max(obstacle(j,i),sor);
        end
    end

    if max(abs(T(:)-Told(:))) < tol
        break
    end
end

fprintf('PSOR iterations = %d\n', iter)

Accuracy, Stability & Convergence Core Theory

A numerical solution is trustworthy only when its discretization is consistent, its algorithm is stable, and the computed sequence converges to the exact PDE solution as the grid is refined.

Deriving finite differences from Taylor series

Taylor expansions around node i
\[ u_{i+1}=u_i+h u'_i+\frac{h^2}{2}u''_i+\frac{h^3}{6}u'''_i+\frac{h^4}{24}u^{(4)}_i+\cdots \] \[ u_{i-1}=u_i-h u'_i+\frac{h^2}{2}u''_i-\frac{h^3}{6}u'''_i+\frac{h^4}{24}u^{(4)}_i+\cdots \]
+
Subtract the two series to eliminate even derivatives: \(u'_i=(u_{i+1}-u_{i-1})/(2h)+O(h^2)\).
Σ
Add the two series to eliminate odd derivatives: \(u''_i=(u_{i-1}-2u_i+u_{i+1})/h^2+O(h^2)\).
Consistency
Discrete equation approaches the PDE
The local truncation error must vanish as \(\Delta x,\Delta t\to0\).
Stability
Errors do not grow without bound
Roundoff, initial-data, and truncation perturbations remain controlled during marching or iteration.
Convergence
Numerical solution approaches exact solution
For well-posed linear initial-value problems, consistency plus stability implies convergence.

Von Neumann stability for FTCS heat conduction

1Assume a Fourier error mode
Let the error be \(e_i^n=G^n e^{\mathrm{i}k i\Delta x}\), where \(G\) is the amplification factor.
2Insert into the FTCS update
\[G=1-4r\sin^2\left(\frac{k\Delta x}{2}\right)\]
3Require every mode to remain bounded
The condition \(|G|\le1\) for every wave number gives \(0\le r\le1/2\).

Boundary-Condition Discretization Essential Skill

Many incorrect PDE solutions come from boundary implementation rather than the interior stencil. These formulas show how to translate physical boundary laws into algebraic equations.

Specified value
\[T(0,t)=T_L,\qquad T(L,t)=T_R\]

Assign these values directly at every time level. Their contributions move to the right-hand side of adjacent interior equations.

Specified flux at x = 0
\[-k\frac{\partial T}{\partial x}=q''\]

First-order one-sided form:

\[\frac{T_1-T_0}{\Delta x}=-\frac{q''}{k}\quad\Rightarrow\quad T_0=T_1+\frac{q''\Delta x}{k}\]

Second-order one-sided form:

\[\frac{-3T_0+4T_1-T_2}{2\Delta x}=-\frac{q''}{k}\]
Convection boundary
\[-k\frac{\partial T}{\partial x}=h_c(T_s-T_\infty)\]

Using a first-order outward derivative at the right boundary:

\[-k\frac{T_N-T_{N-1}}{\Delta x}=h_c(T_N-T_\infty)\]

Therefore:

\[T_N=\frac{(k/\Delta x)T_{N-1}+h_cT_\infty}{k/\Delta x+h_c}\]
Zero-gradient symmetry plane
\[\left.\frac{\partial T}{\partial n}\right|_{\text{sym}}=0\]

A ghost-node central approximation gives \(T_{-1}=T_1\). At a symmetry boundary, the second derivative becomes

\[T_{xx}\big|_0\approx\frac{2(T_1-T_0)}{\Delta x^2}\]
🌡️
Worked Boundary Example — Fin with Convection Tip

A fin has conductivity \(k=15\,\mathrm{W/(m\,K)}\), \(\Delta x=0.01\,\mathrm{m}\), tip convection coefficient \(h_c=30\,\mathrm{W/(m^2K)}\), ambient \(T_\infty=25^\circ\mathrm C\), and the last interior value \(T_{N-1}=80^\circ\mathrm C\). Estimate the tip temperature.

1Insert the Robin boundary data
\[T_N=\frac{(15/0.01)(80)+30(25)}{15/0.01+30}=\frac{120750}{1530}=78.9216^\circ\mathrm C\]
2Interpretation
The small tip temperature drop is reasonable because conduction conductance \(k/\Delta x=1500\) is much larger than convection coefficient \(30\).

Advection–Diffusion–Reaction PDEs Chemical Engineering

Transport in reactors, porous media, membranes, and environmental systems often combines convection, molecular diffusion, and chemical reaction.

General one-dimensional transport model
\[\frac{\partial C}{\partial t}+v\frac{\partial C}{\partial x}=D\frac{\partial^2C}{\partial x^2}+R(C)\]

For a first-order sink, \(R(C)=-kC\).

Cell Peclet number
\[Pe_h=\frac{|v|\Delta x}{D}\]
Diffusion number
\[r_D=\frac{D\Delta t}{\Delta x^2}\]
Courant number
\[Co=\frac{|v|\Delta t}{\Delta x}\]
Reaction number
\[Da_t=k\Delta t\]

Central\((C_{i+1}-C_{i-1})/(2\Delta x)\)2Accurate when diffusion is strong; may oscillate at high \(Pe_h\)

Worked Example — One Explicit Upwind Step

Let \(v=0.2\), \(D=0.01\), \(k=0.1\), \(\Delta x=0.1\), \(\Delta t=0.02\), and local values \(C_{i-1}^n=1.0\), \(C_i^n=0.8\), \(C_{i+1}^n=0.5\). Compute \(C_i^{n+1}\) using upwind advection and central diffusion.

1Compute nondimensional coefficients
\[Co=0.04,\qquad r_D=0.02,\qquad Da_t=0.002\]
2Write the explicit update
\[C_i^{n+1}=C_i^n-Co(C_i^n-C_{i-1}^n)+r_D(C_{i-1}^n-2C_i^n+C_{i+1}^n)-Da_tC_i^n\]
3Substitute values
\[C_i^{n+1}=0.8-0.04(-0.2)+0.02(-0.1)-0.002(0.8)=0.8044\]
▶ MATLAB Code — Advection–Diffusion–Reaction
Upwind transport solver
%% 1D advection-diffusion-reaction equation
clc; clear; close all;

L = 1; Nx = 101; x = linspace(0,L,Nx)';
dx = x(2)-x(1);
v = 0.2; D = 0.01; k = 0.1;
dt = 0.4*min(dx/abs(v),dx^2/(2*D));
Nt = 600;

C = exp(-180*(x-0.25).^2);
C(1) = 0; C(end) = 0;

for n = 1:Nt
    Cold = C;
    for i = 2:Nx-1
        adv = -v*(Cold(i)-Cold(i-1))/dx;  % upwind for v > 0
        dif = D*(Cold(i-1)-2*Cold(i)+Cold(i+1))/dx^2;
        rxn = -k*Cold(i);
        C(i) = Cold(i) + dt*(adv+dif+rxn);
    end
    C([1 end]) = 0;
end

plot(x,C,'LineWidth',2); grid on;
xlabel('x'); ylabel('C');
title('Advection-Diffusion-Reaction Solution');

Verification, Validation & Grid Independence Trust the Result

Verification asks whether the equations were solved correctly. Validation asks whether the equations represent the real physical system adequately.

Code / solution verification
Compare against an analytical solution, manufactured solution, benchmark, or a finer-grid result.
Model validation
Compare model predictions with experiments and quantify measurement and parameter uncertainty.
Maximum norm
\[\|e\|_\infty=\max_i|u_i-u_i^{exact}|\]
Discrete L2 norm
\[\|e\|_2=\sqrt{\frac1N\sum_i e_i^2}\]
Observed order
\[p=\frac{\ln(E_h/E_{h/2})}{\ln 2}\]
Richardson extrapolation
\[u_{ext}\approx u_{h/2}+\frac{u_{h/2}-u_h}{2^p-1}\]

Worked Example — Determine Observed Order

A numerical method gives errors \(E_{0.1}=1.84\times10^{-3}\), \(E_{0.05}=4.61\times10^{-4}\), and \(E_{0.025}=1.15\times10^{-4}\). Determine the observed order.

1First refinement pair

2Second refinement pair
\[p_2=\frac{\ln(4.61\times10^{-4}/1.15\times10^{-4})}{\ln2}=2.003\]
3Conclusion
The observed order is approximately 2, matching a second-order spatial discretization.

P1Derive the five-point stencil for \(u_{xx}+u_{yy}=f(x,y)\) on a nonuniform rectangular grid.
Skills: Taylor expansion, unequal spacing, source termadvanced
Open full solution

Let west and east spacings be \(h_W=x_i-x_{i-1}\) and \(h_E=x_{i+1}-x_i\). A conservative approximation, second-order on uniform or smoothly varying grids, is

\[u_{xx}\big|_{i,j}\approx\frac{2}{h_W+h_E}\left(\frac{u_E-u_P}{h_E}-\frac{u_P-u_W}{h_W}\right)\]

Similarly, with \(h_S\) and \(h_N\),

\[u_{yy}\big|_{i,j}\approx\frac{2}{h_S+h_N}\left(\frac{u_N-u_P}{h_N}-\frac{u_P-u_S}{h_S}\right)\]

Set the sum equal to \(f_{i,j}\), collect coefficients, and obtain

\[a_Pu_P=a_Eu_E+a_Wu_W+a_Nu_N+a_Su_S-f_{i,j}\] \[a_E=\frac{2}{h_E(h_W+h_E)},\quad a_W=\frac{2}{h_W(h_W+h_E)},\quad a_N=\frac{2}{h_N(h_S+h_N)},\quad a_S=\frac{2}{h_S(h_S+h_N)}\] \[a_P=a_E+a_W+a_N+a_S\]

For equal spacing \(h\), this reduces to \((u_E+u_W+u_N+u_S-4u_P)/h^2=f_P\).

P2Construct and solve the 3×3 interior Laplace system for a plate with top = 100°C and other sides = 0°C.
Skills: node numbering, sparse matrix, symmetryadvanced
Open full solution

Use row-wise numbering \(p=i+(j-1)3\), with \(j=1\) at the bottom. Each interior equation is \(4T_P-T_E-T_W-T_N-T_S=b_P\). The vector is \(\mathbf T=[T_{11},T_{21},T_{31},T_{12},\ldots,T_{33}]^T\).

\[ A=\begin{bmatrix} 4&-1&0&-1&0&0&0&0&0\\ -1&4&-1&0&-1&0&0&0&0\\ 0&-1&4&0&0&-1&0&0&0\\ -1&0&0&4&-1&0&-1&0&0\\ 0&-1&0&-1&4&-1&0&-1&0\\ 0&0&-1&0&-1&4&0&0&-1\\ 0&0&0&-1&0&0&4&-1&0\\ 0&0&0&0&-1&0&-1&4&-1\\ 0&0&0&0&0&-1&0&-1&4 \end{bmatrix},\quad b=\begin{bmatrix}0&0&0&0&0&0&100&100&100\end{bmatrix}^T \]

Solving gives approximately

\[ \begin{bmatrix} 7.1429&9.8214&7.1429\\ 18.7500&25.0000&18.7500\\ 42.8571&52.6786&42.8571 \end{bmatrix} \]

The field is symmetric about the vertical centerline and increases toward the hot boundary.

P3Design the largest stable explicit time step for a steel slab and compute the first update.

\(\alpha=1.2\times10^{-5}\,\mathrm{m^2/s}\), \(L=0.02\,\mathrm m\), \(N_x=20\), boundaries 200°C and 20°C, initial interior 20°C. Use a 10% safety margin.

Open full solution

Grid spacing: \(\Delta x=L/N_x=0.001\,\mathrm m\).

The FTCS limit is \(\Delta t_{max}=\Delta x^2/(2\alpha)=10^{-6}/(2.4\times10^{-5})=0.0416667\,\mathrm s\).

With a 10% safety margin, choose \(\Delta t=0.9\Delta t_{max}=0.0375\,\mathrm s\). Then \(r=0.45\).

At the first interior node:

\[T_1^1=0.45(200)+(1-0.9)(20)+0.45(20)=101^\circ\mathrm C\]

All other interior nodes remain 20°C after the first step because their three old-level values are initially equal.

P4Compare one Backward Euler and one Crank–Nicolson step for a high Fourier number \(r=8\).
Open full solution

Both schemes are unconditionally stable for the linear heat equation, but their amplification factors differ:

\[G_{BE}=\frac{1}{1+4r\sin^2(\theta/2)},\qquad G_{CN}=\frac{1-2r\sin^2(\theta/2)}{1+2r\sin^2(\theta/2)}\]

For the highest-frequency mode, \(\sin^2(\theta/2)=1\):

\[G_{BE}=\frac1{33}=0.0303,\qquad G_{CN}=\frac{1-16}{1+16}=-0.8824\]

Backward Euler strongly damps the high-frequency error. Crank–Nicolson keeps a large magnitude and flips its sign, so nonphysical temporal oscillation may appear. A practical remedy is two initial Backward Euler half-steps (Rannacher smoothing) before switching to Crank–Nicolson.

P5Discretize a steady advection–diffusion equation and determine when central differencing may oscillate.

For \(vC_x=DC_{xx}\), derive the central-difference equation and identify the coefficient positivity condition.

Open full solution
\[v\frac{C_E-C_W}{2\Delta x}=D\frac{C_W-2C_P+C_E}{\Delta x^2}\]

After collecting terms:

\[\left(\frac{2D}{\Delta x^2}\right)C_P=\left(\frac{D}{\Delta x^2}+\frac{v}{2\Delta x}\right)C_W+\left(\frac{D}{\Delta x^2}-\frac{v}{2\Delta x}\right)C_E\]

To keep the east coefficient nonnegative for \(v>0\):

\[\frac{D}{\Delta x^2}-\frac{v}{2\Delta x}\ge0\Rightarrow \frac{v\Delta x}{D}\le2\]

Thus central differencing becomes prone to oscillations when the cell Peclet number exceeds about 2. Upwind differencing restores coefficient positivity at the cost of numerical diffusion.

P6Implement a convective boundary in a 1D implicit heat equation matrix.
Open full solution

For the right boundary, \(-k(T_N-T_{N-1})/\Delta x=h_c(T_N-T_\infty)\), define \(Bi_h=h_c\Delta x/k\). Then

\[(1+Bi_h)T_N-T_{N-1}=Bi_hT_\infty\]

This equation replaces the usual interior finite-difference row. If the boundary node is eliminated instead,

\[T_N=\frac{T_{N-1}+Bi_hT_\infty}{1+Bi_h}\]

Substitute this relation into the adjacent interior-node equation, changing both its diagonal coefficient and right-hand side.

P7Estimate the optimal SOR relaxation factor for a 50×50 square Laplace grid.
Open full solution

For a standard square grid, a useful estimate is

\[\omega_{opt}\approx\frac{2}{1+\sin(\pi/(N+1))}\]

For \(N=50\):

\[\omega_{opt}\approx\frac{2}{1+\sin(\pi/51)}=1.884\]

This is an estimate, not a universal guarantee. Geometry, variable coefficients, ordering, and boundary conditions can shift the best value.

P8Perform a grid-convergence study and Richardson extrapolation.

At a monitored point, solutions are \(u_h=0.8421\), \(u_{h/2}=0.8556\), and \(u_{h/4}=0.8590\). Estimate the order and extrapolated value.

Open full solution

For a constant refinement ratio \(r=2\),

\[p=\frac{\ln\left|\frac{u_h-u_{h/2}}{u_{h/2}-u_{h/4}}\right|}{\ln2}=\frac{\ln(0.0135/0.0034)}{\ln2}=1.989\]

Use \(p\approx1.989\) in Richardson extrapolation:

\[u_{ext}=u_{h/4}+\frac{u_{h/4}-u_{h/2}}{2^p-1}\approx0.8590+\frac{0.0034}{2^{1.989}-1}=0.86014\]

The finest-grid estimate is about \(0.13\%\) below the extrapolated value.

P9Derive the first time level for the wave equation with nonzero initial velocity.
Open full solution

The central wave update needs \(u^{-1}\), which is not given. Use the initial velocity \(g_i=u_t(x_i,0)\):

\[\frac{u_i^1-u_i^{-1}}{2\Delta t}=g_i\Rightarrow u_i^{-1}=u_i^1-2\Delta t g_i\]

Apply the PDE stencil at \(n=0\):

\[u_i^1=2u_i^0-u_i^{-1}+\lambda^2(u_{i-1}^0-2u_i^0+u_{i+1}^0)\]

Substitute the velocity relation and solve for \(u_i^1\):

\[u_i^1=u_i^0+\Delta t g_i+\frac{\lambda^2}{2}(u_{i-1}^0-2u_i^0+u_{i+1}^0)\]
P10Select a complete numerical strategy for a 2D exothermic catalytic pellet model.
Open full solution

A coupled concentration–temperature model may be written as

\[\frac{\partial C}{\partial t}=D_{eff}\nabla^2C-r(C,T),\qquad \rho c_p\frac{\partial T}{\partial t}=k_{eff}\nabla^2T+(-\Delta H)r(C,T)\]

Recommended strategy:

  1. Use a body-fitted finite-volume or finite-element mesh for pellet geometry.
  2. Use second-order diffusion discretization and physically consistent symmetry plus film-transfer Robin boundaries.
  3. Apply Method of Lines with a stiff integrator such as ode15s or a fully implicit Newton method.
  4. Assemble an analytic or sparse numerical Jacobian because Arrhenius kinetics can be strongly nonlinear.
  5. Scale concentration and temperature unknowns to reduce conditioning problems.
  6. Verify against an isothermal Thiele-modulus solution, then perform temporal and spatial refinement.
  7. Check total mass and energy balances and inspect whether multiple steady states are physically possible.
